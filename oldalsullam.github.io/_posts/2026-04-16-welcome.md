    <h1>Welcome to AL Sullam</h1>

    <p><strong>AL Sullam</strong> (السُّلَّم — "The Stair" in Arabic) is a systems programming organization with a simple mission: build software from first principles.</p>

    <p>We believe that understanding comes from building. Not from reading documentation, not from using frameworks, but from implementing things from scratch — byte by byte, line by line.</p>

    <h2>What We Build</h2>

    <p>Our projects rebuild fundamental layers of computing:</p>

    <ul>
        <li><strong>BitTorrent</strong> — A complete P2P client in C with no external dependencies</li>
        <li>More to come...</li>
    </ul>

    <p>Every project is fully documented with chapters that explain the theory, the algorithm, and the implementation — because code without explanation is archaeology.</p>

    <h2>Our Principles</h2>

    <ol>
        <li><strong>No Black Boxes</strong> — If we use it, we could write it</li>
        <li><strong>Document Everything</strong> — Explain the why, not just the how</li>
        <li><strong>The Spec Is the Authority</strong> — Read RFCs, BEPs, ISO standards</li>
        <li><strong>Low-Level First</strong> — Start in C where nothing is hidden</li>
        <li><strong>Correctness Over Performance</strong> — Make it right before making it fast</li>
    </ol>

    <h2>What's Next</h2>

    <p>We're just getting started. More projects are in development, more chapters are being written, and more of the stack is being rebuilt from scratch.</p>
